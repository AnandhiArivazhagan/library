# library
MERN stack application for a book reading list
Tech Stack: HTML5, CSS3, JavaScript, Node.js, Express.js, MongoDB
•	Developed a full-stack digital book library where users can add, categorize, mark as read, and delete books through a clean interface.
•	Implemented CRUD operations with MongoDB for persistent book data storage and real-time UI updates.
•	Built a category-based organization system allowing users to filter and manage books by genre or reading preference.
•	Designed an intuitive read/unread tracking feature to help users monitor their reading progress and wishlist.
